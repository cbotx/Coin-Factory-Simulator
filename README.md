## Coin Factory Simulator
A simulator of the game Coin Factory for designing and optimizing strategies
![image](./images/ui.png)

### Requirements
[.Net 9.0](https://dotnet.microsoft.com/en-us/download/dotnet/9.0)

### Controls
#### Frames
Arrow keys: Navigation

Ctrl + ← / Ctrl + →: Navigation

Right click: Remove a frame

\+ / \- : Add / remove an empty frame

#### Play area
Middle click: Upgrade a block

Right click: Remove a block

Scroll: Rotate a block

#### In-game execution
While in the game:

Tab (No need to focus): Next keyframe
